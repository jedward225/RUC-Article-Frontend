AI写作反馈逻辑树（修订）

总体目标：
通过AI标注识别学生段落在结构、逻辑与论据层面的表现，以PEEL结构 →论据类型连接词→词汇句型使用 的顺序提供层层递进的反馈，支持学生理解反馈、决策修订并形成循环改进。

Level 1：宏观结构层（PEEL 段落反馈）
模块 1：【PEEL结构反馈】
系统功能：综合Ev（论据）与T（连接词）标注，诊断段落是否具备完整的PEEL结构（Point–Evidence–Explanation–Link）。系统仅判断是否“出现/未出现”PEEL元素，不评估质量或逻辑有效性。

反馈逻辑与对应反馈示例：
PEEL	系统检测	反馈逻辑
（只判断有无）	反馈示例
（不做质量判断）
P – Point（主题句 / 中心论点）	系统是否检测到主张性句（主题句模式）	若未检测到 → 提示“结构不完整”	“段落未使用明确的主题句。建议在段首用一句话清晰表明你的立场。”
E – Evidence（论据支持）	是否检测到任意 Ev 标签（eg / quo / dat / reason 任意一种）	若未检测到 → 提示“缺少支撑”	“段落未使用论据支持。可加入例证、引用或数据来支撑你的观点。”
E – Explanation（解释句 / 推理句）	是否检测到解释句	若未检测到 → 提示“缺少解释”	“段落未使用解释性句子。可加入一句说明论据与观点之间关系的解释句。”
L – Link（总结句 / 过渡句）	段末是否检测到总结、过渡或回扣结构	若未检测到 → 提示“缺少收束”	“段末未使用总结或过渡表达。如有需要，可加入一句总结或引出下一段的句子。”

建议：在首句明确立场，在例证后补充解释性语句，以总结句收束段落。
学生任务：Check理解 → 是否修订段落结构。

Level 2：内容支撑层 
模块 2：【论据类型反馈】
系统功能：识别Ev类型（Ev-eg, Ev-quo, Ev-dat, Ev-reason）并检测论据多样性。
反馈逻辑：
本段落检测到以下论据类型：
oEv-eg（例证）
oEv-reason（原因说明）
（注：系统仅识别类型，不判断质量或恰当性。）
系统反馈（只做元认知提醒，不做判断）：
该段目前主要依赖 Ev-eg（例证） 与 Ev-reason（原因说明）。若你希望增强段落的可信度或深度，可考虑加入 引用（Ev-quo） 或 数据（Ev-dat） 类型的支撑。
（教师可补充：各类型常用用途、语篇功能）
学生任务：
查看系统列出的论据类型
判断是否需要增加类型多样性
决定是否补充引用或数据型论据

Level 3：衔接层

模块 3：【连接词反馈】
系统功能：识别并分类段落中的逻辑连接词（T-add / T-cau / T-seq / T-adv 等）。
反馈逻辑：
情况 A：未检测到任何连接词
系统反馈：
你的段落中并未使用任何连接词。建议回到文本，检查句子之间的逻辑关系是否清晰，可以考虑加入表示 递进 / 因果 / 顺序 / 转折 的连接词。
学生任务：
请检查你的句子之间的逻辑关系是否明确
视情况判断是否需要加入合适的连接词

情况 B：检测到连接词（T detected）
系统展示：
本段落检测到以下连接词类型：
T-add（递进类）：___
T-cau（因果类）：___
T-seq（顺序类）：___
T-adv（转折类）：___
（注：系统仅呈现使用情况，不判断是否恰当）
系统反馈（仅提醒，不判定对错）：
请根据自己的语义意图判断：这些连接词是否与句子之间的逻辑关系相符合？
如果你发现逻辑关系不属于当前连接词类型，可以考虑调整连接方式。
学生任务：
查看系统展示的连接词类别
思考这些类别是否与自己表达的句间逻辑一致
决定是否进行修订


Level 4：语言层（语法与词汇诊断）
模块 4：【语言诊断】
功能定位：由外部语言模型介入，对修订后段落进行微观层面的语言评估与精准修正。
诊断维度	外部模型功能	示例反馈
语法准确性	自动检测主谓一致、时态、冠词、从句结构	“句中 ‘have discussed’ 应改为 ‘has discussed’。”
词汇适切性	检测词汇搭配、语义强度、学术风格	“‘very big’ 可替换为 ‘substantial’ 以符合学术语体。”
句式多样性	分析句型长度与复杂度分布	“多数句为简单句，建议增加从句或过渡结构。”
模型调用逻辑：
–该模块独立调用外部大模型API；
–接受来自上一步修订后的文本；
–输出校订结果与说明（可选是否直接替换文本）。
学生任务：Check理解（确认模型修改是否合理）；决定是否采纳修订建议

Level 5：反馈环节结束
模块 5：【综合反馈汇总】
肯定学生已完成的部分（如：已出现论点、使用了连接词、加入了论据等）
鼓励学生继续关注结构与逻辑清晰度
强调修订是写作过程的一部分
提供积极、成长性的鼓励语

【你已在段落中呈现了若干关键写作要素，例如：明确观点、使用部分论据类型、加入一定的连接词等。你的段落已经具备基本的结构。你可以持续关注段落内部各要素的衔接，如论据与观点的逻辑关系、段首与段末的结构呼应等。写作是一个不断修订和完善的过程，能够认真查看反馈并思考如何改进已经非常值得肯定。保持探索与修订的态度，是写作进步的重要动力。继续加油！】
